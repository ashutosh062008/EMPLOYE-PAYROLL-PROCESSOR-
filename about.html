About the System

Technical architecture and design principles of PayrollPro.

🏗️ OOP Architecture

Built using ES6 classes and inheritance. A base Employee class provides polymorphic calculateSalary methods across FullTime, Contract, and Hourly subclasses.

🎨 Modern UI/UX

Featuring a state-of-the-art dark theme with Glassmorphism effects, CSS Grid layouts, and hardware-accelerated micro-animations for a premium desktop experience.

💾 Data Resilience

Powered by Browser localStorage, your payroll records are saved permanently and persist even after closing the browser or refreshing the page.

📊 Advanced Logic

Handles complex payroll rules: hourly overtime at 1.5× after 40 hours, dynamic custom tax rates for contracts, and a benefits deduction model for full-time employees.

Tech Stack

HTML5 CSS3 Vanilla JavaScript ES6 Classes localStorage API CSS Grid / Flexbox Google Fonts Glassmorphism
